## Getting Started

- First, run the development server:

```bash
yarn install
```
- Copy .env.example and Edit .env and set value

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features
- Next.js - TypeScript
- Wagmi
- ChakraUI
- Next Router
